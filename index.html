開始新的專案

  1. 先在local端建立好資料存放位置
  2. 連上https://github.com/點選New repository(repositories)
  3. 填寫專案名稱、敘述、勾選狀態(private需要付費)，點擊create
  4. 對local資料夾按右鍵選擇用vscode開啟，在下方terminal確認路徑正確後，照做下列步驟
  5. git init (初始化)
    git add README.md (可不填)
    git commit -m "first commit" (設定commit名稱並存入changes)
    git branch -M master (設定default分支名稱，後續可修改)
    git remote add origin https://github.com/Ryan-Lan00/123.git (與local資料連結)
    git push -u origin master (設定倉庫名稱(origin)設定後無法更改)
    6. git pull -u origin master (push pull 都先輸入讓他記錄該到哪個分支)

推送編輯

  1. 先儲存後在local端確認compile、回傳資料是否正確
